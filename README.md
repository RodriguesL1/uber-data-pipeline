# 🚀 Uber Data Pipeline

Projeto de pipeline de dados utilizando Python e SQL para análise de corridas Uber.

## 📊 Tecnologias
- Python
- Pandas
- PostgreSQL
- SQL

## 🔄 Pipeline
1. Extração de dados (CSV)
2. Transformação e limpeza
3. Carga no banco de dados
4. Análise com SQL

## 📈 Resultados
- Média de valor das corridas
- Volume por período
- Insights de uso

## ▶️ Como executar

```bash
pip install -r requirements.txt
python src/extract.py
